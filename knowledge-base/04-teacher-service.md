# 교원 복무 관련 규정

> 국가공무원 복무규정 최종 개정: 2025.2.11. (대통령령 제35258호) | 교육공무원법 최종 개정: 2024.9.20.
> 출처: law.go.kr, mpm.go.kr

---

## 1. 근무시간 (국가공무원 복무규정 제9조)

| 항목 | 내용 |
|------|------|
| 1주 근무시간 | **40시간** (점심시간 제외) |
| 1일 근무시간 | **09:00 ~ 18:00** |
| 점심시간 | **12:00 ~ 13:00** (기관장이 변경 가능) |

---

## 2. 연가 (제15조)

### 재직기간별 연가일수

| 재직기간 | 연가일수 |
|----------|----------|
| 1개월~1년 미만 | **11일** |
| 1년~2년 미만 | **12일** |
| 2년~3년 미만 | **14일** |
| 3년~4년 미만 | **15일** |
| 4년~5년 미만 | **17일** |
| 5년~6년 미만 | **20일** |
| 6년 이상 | **21일** |

- 미사용 연가: **연가보상비** 지급 (최대 20일)
- **반일연가**: 오전·오후 구분, 2회 = 1일

---

## 3. 병가 (제18조)

| 구분 | 일수 |
|------|------|
| **일반 병가** | 연 **60일** |
| **공무상 병가** | 연 **180일** |

- **6일까지**: 진단서 불필요
- **7일 이상**: 의사 **진단서 필수**

---

## 4. 공가 (제19조)

- 병역판정검사, 투표, 승진시험 응시, 헌혈, 건강검진, 예방접종 등

---

## 5. 특별휴가 (제20조)

### 경조사휴가

| 구분 | 대상 | 일수 |
|------|------|------|
| **결혼** | 본인 | **5일** |
| | 자녀 | **1일** |
| **입양** | 본인 | **20일** |
| **사망** | 배우자, 부모 | **5일** |
| | 조부모, 외조부모 | **3일** |
| | 자녀, 형제자매 | **3일** |

### 출산 관련

| 구분 | 일수 |
|------|------|
| **출산휴가** | **90일** (미숙아 100일, 다태아 120일) |
| **배우자 출산휴가** | **20일** (2025.2.11. 개정) |

### 기타

| 구분 | 내용 |
|------|------|
| **육아시간** | 만 8세 이하 자녀, 1일 **2시간** (36개월 범위) |
| **가족돌봄휴가** | 연 **10일** |
| **장기재직휴가** (2025 신설) | 10~20년: **5일**, 20년 이상: **7일** |

---

## 6. 41조 연수 (교육공무원법 제41조)

- 수업에 지장 없는 범위에서 **소속기관장 승인** 후 연수
- 주로 **방학 기간** 중 실시
- 연수 계획서 → 교장 승인 → 연수 → 결과 보고

---

## 7. 교원 성과상여금 (2025년)

| 등급 | 배분 비율 |
|------|-----------|
| **S등급** | **30%** |
| **A등급** | **50%** |
| **B등급** | **20%** |

- 차등지급률은 **학교별 자율 결정** (50%~100%)
- 지급 시기: 매년 **3월 말**

---

## 8. 겸직 허가

- **국가공무원법 제64조**: 영리업무 종사 금지 및 겸직 허가
- 외부 강의, 원고 집필 등은 **사전 겸직 허가** 필요
- 유튜브 수익 창출 등은 **금지 대상**일 수 있음

---

## 9. 출장 규정

| 구분 | 기준 | 여비 |
|------|------|------|
| **관내** | 동일 시·군 또는 12km 미만 | 4시간 미만: 1만원, 이상: 2만원 |
| **관외** | 근무지 외 지역 | 일비 2만원 + 식비 2.5만원 + 숙박·교통 실비 |

---

## 10. 전보 기준

- **시도교육청별** 자체 인사관리기준으로 운영
- 동일교 근무 상한: 통상 **5~8년**
- 전보 점수: 근무지역점 + 경력점 + 부가점
- 임신·출산 후 1년 미만 교원 **모성보호 전보 배려**

---

## 초등교사 실무 포인트

1. **수업일 중 연가**: 보결수업 계획 수립 후 사용
2. **병가 7일 이상**: 진단서 필수 (6일까지는 불필요)
3. **배우자 출산휴가 20일**: 2025.2.11.부터 적용, 30일 이내 분할 사용 가능
4. **겸직**: 외부 강의·원고료 수령 시 반드시 **사전 허가 신청**
5. **41조 연수**: 방학 중 사전 승인 필수
